# Remember

Persist current session context to memory for future reference.

## Actions

1. Invoke the `memory` skill
2. Store the following as entities and observations:
   - Tasks completed this session
   - Key decisions made and their reasoning
   - Files created or modified
   - Open questions or blockers
   - Technical learnings or patterns discovered
3. Create relations to existing entities where relevant
4. Use descriptive entity names that will be searchable later

## When to Use

- End of a work session
- After completing a significant task
- Before context might be lost
- When switching to a different project
